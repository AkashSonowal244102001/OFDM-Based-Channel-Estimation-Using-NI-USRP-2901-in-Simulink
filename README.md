# OFDM-Based-2-2-MIMO-Channel-Estimation-Using-NI-USRP-2901-in-Simulink
Designed and implemented an OFDM-based 2×2 MIMO wireless transceiver using NI USRP-2901 in Simulink to experimentally estimate the wireless channel frequency response and channel impulse response (CIR) under real over-the-air conditions.
